# Cursor Obsidian Conversation Package

This package provides one Cursor skill and three focused subagents for turning free-form conversation into Obsidian-compatible raw notes and memory notes.

## Included
- `.cursor/skills/obsidian-conversation-to-memory/SKILL.md`
- `.cursor/skills/obsidian-conversation-to-memory/references/obsidian-output-contract.md`
- `.cursor/skills/obsidian-conversation-to-memory/assets/raw-note-template.md`
- `.cursor/skills/obsidian-conversation-to-memory/assets/memory-note-template.md`
- `.cursor/skills/obsidian-conversation-to-memory/assets/save-memory-v2.schema.json`
- `.cursor/agents/obsidian-metadata-scout.md`
- `.cursor/agents/obsidian-memory-splitter.md`
- `.cursor/agents/obsidian-memory-verifier.md`

## Install
Copy the `.cursor/` folder into your repository root.
