## Section 3: Milestone Dictionary + E2E 프로세스 연결

### 3.1 두 Stage 시스템의 통합

현재 두 가지 단계 체계가 존재합니다. 이를 **역할별로 명확히 분리**합니다:

| 시스템 | 출처 | 역할 | 타입 |
|---|---|---|---|
| **Journey Stage** (15단계) | ROLE_PATCH.MD → CONSOLIDATED-01 삽입됨 | `ShipmentUnit.currentStage` = 현재 상태 버킷 | 상태값 (enum) |
| **Milestone** (M10~M160) | ROLE_PATCH2.MD Section 5.5 | 이벤트 발생 시점 기록 | 이벤트 (planned/estimated/actual) |

**관계**: Milestone 발생 → Journey Stage 전환 트리거

```
M110 (Warehouse Received) 발생
    → ShipmentUnit.currentStage = "WH_RECEIPT" (Stage 7)

M121 (Dispatched) 발생
    → ShipmentUnit.currentStage = "WH_DISPATCH" (Stage 9)
```

### 3.4 E2E 19단계 → Milestone → Journey Stage 매핑

```
Step 1:  PO Release           → (no milestone)    → PLANNING
Step 2:  Vendor Readiness     → M10               → PLANNING
Step 3:  Packing              → M20               → ORIGIN_DISPATCH
Step 4:  Pickup               → M30               → ORIGIN_DISPATCH
Step 5:  Export Clearance     → M40               → ORIGIN_DISPATCH
Step 6:  Terminal Receiving   → M50               → PORT_ENTRY
Step 7:  Vessel Loading/ATD   → M60, M61          → TERMINAL_HANDLING
Step 8:  Ocean / Transship    → M70               → TERMINAL_HANDLING
Step 9:  ATA / Pre-arrival    → M80               → PORT_ENTRY
Step 10: Import BOE           → M90, M91          → CUSTOMS_CLEARANCE
Step 11: DO / Gate-out        → M92, M100         → INLAND_HAULAGE
Step 12: Inland Transport     → (leg event)       → INLAND_HAULAGE
Step 13: WH Receiving     ★  → M110              → WH_RECEIPT
Step 14: Put-away / Storage   → M111              → WH_STORAGE
Step 15: Picking / Dispatch   → M120, M121        → WH_DISPATCH
Step 16: Offshore/Heavy-lift  → (marine event)    → MOSB_STAGING / OFFSHORE_TRANSIT
Step 17: Site Delivery/Insp.  → M130, M131/M132   → SITE_RECEIVING
Step 18: POD / GRN            → M140              → MATERIAL_ISSUE
Step 19: Exception / Cost     → M150, M160        → CLOSEOUT
```

★ Step 13이 `WarehouseHandlingProfile` 생성 트리거

### 3.5 AGI/DAS 도메인 룰과 Milestone 연동

VIOLATION-2 게이트 구체화:

```
IF ShipmentUnit.declaredDestination IN (AGI, DAS)
   AND ShipmentUnit.routingPattern IN (MOSB_DIRECT, WH_MOSB, MIXED)
   AND MilestoneEvent(M130).actualDt IS NOT NULL
   AND MilestoneEvent(MOSB_STAGING).actualDt IS NULL
→ VIOLATION-2: AGI/DAS 화물에 MOSB Milestone 누락
```
